# Library Book System

## What it does
This program contains a book class that allows to check books out and return them. It also contains a method that returns a report of the overdue books and a search function. This are basic functions every library should have.
## How to run it
1. Open the notebook in Google Colab
2. Run all cells from top to bottom

## Skills demonstrated
- Classes
- For-Loops
- If-Statements
- Methods
- Dictionaries
- Variables
- Data Types
- f-Strings

## What it produces
The program produces a simple library system that's able to track checked out books, returns, overdues, and has a simple search funcion.
